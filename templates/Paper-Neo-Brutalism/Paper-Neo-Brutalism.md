## 布局创意 (Layout Ideas)

### Hero 区

- 左侧超大展示标题（92px），字距紧凑（-2.76px），营造杂志封面般的冲击力；标题断行处理，形成强烈视觉锚点。
- 右侧悬浮“系统AI助手”胶囊按钮与信息卡片，建立功能入口与辅助说明的层级。
- 下方主卡片区采用两栏网格：左侧文案+双按钮，右侧项目简介面板，整体像桌面上的便签拼贴。

### 核心入口区（Features）

- 四列等宽卡片网格，每张卡片顶部标有序号（01-04）与 Lucide 风格图标，形成清晰的模块化入口。
- 卡片使用 2px 微圆角、1px 细边框和硬边阴影，呈现可点击的“纸块”质感。
- 标题与描述之间使用紧凑间距，保持信息密度。

### 延展方向区（Tool / Workflow）

- 左侧为交互式工具卡片（押金计算器），包含表单输入、下拉选择与结果展示；右侧为带编号步骤的列表。
- 整体保持双栏网格，但在移动端应折叠为单列。

### 导航与页脚

- 顶部固定导航栏使用半透明毛玻璃背景（blur 16px），Logo + 文字链接 + 图标按钮横向排列。
- 顶部横幅条为深墨底配亮绿色点缀，展示关键提示信息。

## 完整提示词 (Full Prompt)

# 设计风格：纸质现代主义（Paper Neo-Brutalism）

## 设计哲学

### 核心原则

- **纸质为底**：以米色、奶油色、暖白色模拟纸张表面，拒绝冰冷纯白背景，追求可触摸的实体感。
- **墨印文字**：使用深墨黑（#151511）作为主文字色，保证高对比度的阅读体验，体现印刷品的权威感。
- **绿色点睛**：仅在核心行动点、标签、高亮提示与关键数据处使用亮绿色（#08D36C），形成强烈的视觉指引。
- **硬边阴影**：采用纯黑或同色偏移的硬阴影（4px 4px 0px），替代传统弥散阴影，产生新粗野主义的“贴纸”效果。
- **网格纹理**：在背景中叠加极淡的网格线渐变，暗示工程设计图纸与草稿纸的质感。

### 视觉气质

**情感关键词**：
- *可信赖的* — 高对比度文字、结构化网格、清晰的模块边界，让用户感到专业且安全。
- *手作感的* — 暖色纸张、硬边阴影、2px 微圆角，营造经过精心排版的纸质产品手册感。
- *克制的活力* — 亮绿色点缀在沉稳的纸墨基调中，像荧光笔标记一样精准而不过度。

**此设计不是什么**：
- 不是冷色调的科技风（不追求深蓝、霓虹、玻璃拟态）。
- 不是圆润柔和的 SaaS 通用风格（圆角不超过 2px，不使用大弥散阴影）。
- 不是极简到空白的风格（允许并鼓励网格纹理、阴影、边框等装饰性细节）。

### 此风格的 DNA

#### 1. 暖米色纸纹背景
页面主背景使用 `#f4f1e8`（纸米）与 `#fffdf7`（奶油白）的交替层级，卡片背景为 `#fbfaf5`，模拟不同厚度纸张的叠加。所有表面拒绝纯白，保持温暖、低饱和的纸张质感。

#### 2. 硬偏移阴影卡片
卡片与按钮使用 `box-shadow: #050505 4px 4px 0px 0px` 或 `#08D36C 4px 4px 0px 0px` 的硬边阴影，阴影不做模糊，呈现印刷裁切与贴纸堆叠的立体感。悬停时阴影可略微放大或改变颜色。

#### 3. 超大展示标题与紧凑字距
Hero 标题使用 92px、字重 400、字距 -2.76px 的 DM Sans，标题故意断行形成强烈的块状文字轮廓。标题区不追求装饰图形，让文字本身成为视觉主体。

#### 4. 亮绿色“荧光标记”
行动按钮、重要标签、顶部提示条、图标 hover 状态使用 `#08D36C` 亮绿色。绿色以实心填充、细边框、内阴影或发光阴影的形式出现，像荧光笔划过纸面一样精准。

#### 5. 工程网格纹理
在背景与卡片表面叠加 `linear-gradient(...)` 形成的极淡网格线（透明度约 0.02–0.04），为页面增加“设计图纸”的理性气质，同时不干扰内容阅读。

- **区块间距**：主工作区使用 `padding: 46px 48px 80px`，顶部与底部留白慷慨，内容区紧凑。

**常见间距值**：4px / 8px / 12px / 16px / 24px / 32px / 48px / 80px

### 边框、表面与阴影

**圆角策略**：
- 普通卡片与按钮：`2px`（微圆角，保留纸张裁切感）。

## "大胆要素"（签名元素）

1. **墨黑主按钮配绿色硬阴影**：将传统“黑底白字”按钮用 `#08d36c` 的 4px 硬阴影托出，形成强烈的贴纸感与行动号召力。

2. **92px 超大标题与断行排版**：Hero 标题不依赖插图，仅凭超大字号与紧凑字距营造杂志封面感，文字断行形成几何块面。

3. **奶油纸背景 + 工程网格线**：在暖色纸纹上叠加几乎不可见的 1px 网格线，让页面既温暖又理性，像高级产品手册页。

4. **绿色“荧光标记”式焦点态**：输入框聚焦、按钮悬停使用绿色发光阴影（`0px 0px 18px rgba(8,211,108,0.72)`），模拟荧光笔在纸上的高亮效果。

5. **2px 微圆角与硬边框**：所有卡片、按钮、输入框几乎不使用大圆角，保持纸张裁切的利落感，与弥散阴影的潮流形成反差。

6. **编号卡片网格（01–04）**：核心功能区采用四列编号卡片，每张卡片顶部有序号、图标与标题，像产品目录一样清晰可扫。

## 效果与动画

**运动哲学**：整体动画克制、快速、有“弹纸感”，使用自定义弹簧缓动曲线，强调即时反馈而非装饰性演出。

**过渡默认**：
- 标准：`transform 0.16s cubic-bezier(0.16, 1, 0.3, 1)`
- 复合：`transform 0.16s cubic-bezier(0.16, 1, 0.3, 1), box-shadow 0.16s, background-color 0.16s, color 0.16s`
- 背景/边框：`background 0.18s, border-color 0.18s, transform 0.18s`

**悬停效果**：
- 按钮/卡片：整体 `translate(-2px, -2px)`，阴影同步偏移，营造被“拿起”的纸块感。
- 链接：颜色过渡为 `#08d36c`，或出现绿色下划线。

**持续动画**：
- 无持续动画；仅保留 `spin` 关键帧（用于加载图标等场景）。

**关键帧动画**：
```css
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```

## 响应式策略

**断点哲学**：以桌面端 1440px 为基准展示完整编辑布局，在平板与移动端将多列网格折叠为单列，保持卡片垂直堆叠与阴影效果。

- **Hero 区**：桌面端两栏（标题左 + 信息右），移动端变为单列，标题字号从 92px 降至 48px–36px。
- **四列核心入口**：平板端变为 2×2 网格，移动端变为单列堆叠。
- **工具/延展区**：桌面端双栏（工具左 + 步骤右），移动端单列，步骤列表占据全宽。
- **导航**：桌面端横向展开，移动端折叠为汉堡菜单或底部标签栏，保持毛玻璃背景。

**关键适配**：
- 标题尺寸按视口缩放：Hero 92px → 54px → 36px。
- 区块间距按 48px / 32px / 24px 递减。
- 装饰网格线在移动端可降低透明度或隐藏，避免在小屏上产生噪点。
- 阴影保持 4px 偏移，但可在移动端减为 2px 以节省空间。

## 可访问性与最佳实践

**色彩对比**：
- 主文字 `#151511` 在 `#f4f1e8` 背景上对比度极高，符合 WCAG AA 及以上标准。
- 亮绿色 `#08d36c` 用于文字或图标时需注意与背景的搭配，避免在浅色背景上单独使用小字号绿色文字。
- 墨黑底 `#050505` 上的白色文字对比度充足。

**聚焦状态**：
- 使用绿色发光环（`box-shadow: 0 0 0 4px rgba(8,211,108,0.13), 0 0 18px rgba(8,211,108,0.72)`）作为焦点指示，明显且符合品牌色。
- 链接悬停使用颜色变化 + 下划线，不只依赖颜色变化。

**触控目标**：
- 按钮与导航链接最小触控目标不小于 44×44px，移动端适当增加内边距。
- 卡片整块可点击时，确保焦点范围覆盖整个卡片。

**运动**：
- 遵循 `prefers-reduced-motion: reduce` 时关闭位移动画与缩放，仅保留必要的颜色/透明度过渡。
- 加载动画使用 `spin` 旋转，避免过度闪烁。

## 实现说明

**组件结构**：
- 推荐框架：React + Vite + Tailwind CSS。
- 卡片、按钮、输入框使用统一的 `rounded-sm`（2px）与 `border` 样式。
- 使用 CSS 自定义属性定义颜色与阴影 Token，便于主题切换与复用。

**CSS 自定义属性**：
```css
:root {
  --bg-paper: #f4f1e8;
  --bg-soft: #fbfaf5;
  --bg-cream: #fffdf7;
  --bg-ink: #050505;
  --ink: #151511;
  --muted-ink: #6f6b60;
  --line: #ded8ca;
  --line-strong: #171713;
  --green: #08d36c;
  --green-dark: #00a955;
  --space-xs: 4px;
  --space-sm: 8px;
  --space-md: 12px;
  --space-lg: 16px;
  --space-xl: 24px;
  --space-2xl: 32px;
  --radius-sm: 2px;
  --radius-md: 2px;
  --radius-lg: 2px;
  --ease-out: cubic-bezier(0.16, 1, 0.3, 1);
  --ease-spring: cubic-bezier(0.16, 1, 0.3, 1);
}
```

**字体加载**：
- 使用 Google Fonts 或本地加载 `DM Sans` 的 400、700、800 字重。
- 中文环境回退至 `Microsoft YaHei` / `PingFang SC`。

**动画库**：
- 无需额外动画库，仅使用 CSS transition 与 `@keyframes spin`。
- 入场动画使用 CSS `transform` + `opacity` 自定义过渡，缓动使用 `cubic-bezier(0.16, 1, 0.3, 1)`。
