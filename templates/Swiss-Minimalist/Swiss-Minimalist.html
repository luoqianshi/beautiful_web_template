02. System

The Grid Is Law

Every visual decision is justifiable by the content's needs. Personal ornamentation is eliminated in favor of functional communication. Type is the interface.

01

Mathematical Grids

24px baseline grids with visible structure. Asymmetrical organization creates dynamic visual rhythm and tension without sacrificing legibility.

→
02

Neutral Grotesque Type

Inter at weights 400 to 900. Scale, weight, and position are the only tools needed to create hierarchy. Type recedes so meaning can emerge.

→
03

Active Negative Space

White space is not empty. It is a structural element that defines boundaries, gives weight to massive typography, and creates breathing room.

→
04

Layered Texture Depth

Subtle CSS patterns — grids, dot matrices, diagonals, noise — add tactile richness without compromising the objective, flat aesthetic.

→
01
— STEP ONE

Define The Grid

Establish the modular grid before a single character is set. The structure dictates every subsequent decision, from column widths to type scale.

02
— STEP TWO

Set The Type

Choose neutral grotesque typefaces. Establish a mathematical scale with clear ratios. Flush-left, ragged-right. Uppercase for hierarchy.

03
— STEP THREE

Apply The Signal

Swiss Red (#FF3000) appears only as a functional signal — a stop sign, a warning, a highlight. Never decorative. Always purposeful.

04. Journal

Frequently Asked Questions

01. Why no rounded corners?
Rectangular forms are honest. They reveal the grid and the construction. Softening corners would compromise the objective, architectural character of the system.
02. Is red ever decorative?
Never. Swiss Red is a functional signal — used for CTAs, hover states, and section number prefixes. Decorative use would dilute its meaning as a stop sign.
03. Why Inter and not Helvetica?
Inter is a neutral grotesque with high x-heights, optimized for screens. It carries the same objective DNA as Helvetica while being freely available and screen-optimized.
04. What about shadows and gradients?
None. Depth comes from pattern overlays — grids, dots, diagonals, noise — never from drop shadows or gradients. The design is flat yet rich, stark yet nuanced.
