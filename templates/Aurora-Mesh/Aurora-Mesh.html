Features

Everything you need to ship beautiful UI

A complete system of components, primitives, and design tokens that work together to create interfaces with dimensional depth.

Living mesh gradients

Backgrounds that flow and morph with organic movement. Built on layered radial gradients with GPU-accelerated animation, our mesh system creates the ethereal aurora effect that makes your product feel alive.

Glass surfaces

Backdrop blur with saturation creates true depth and dimension.

Gradient text

Headlines that glow with multi-stop color transitions.

Micro-interactions that feel alive

Every element responds with smooth, considered motion. Hover states glow, cards lift with colored shadows, and transitions flow like water through space. Never jarring, always elegant.

Design tokens

A complete token system for colors, radius, shadows.

Dark by default

Near-black canvas makes aurora colors truly sing.
