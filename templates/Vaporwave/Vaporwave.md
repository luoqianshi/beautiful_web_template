## 完整提示词 (Full Prompt)

# Vaporwave / Outrun 设计系统

## 1. 设计哲学

**"数字怀旧遇上霓虹未来——一个浸泡在复古未来主义过度中的合成现实。"**

这是对 1980 年代复古未来主义、vaporwave 美学与早期计算机图形的大胆赞颂。设计将用户传送至合成数字维度，霓虹光穿透无限网格，CRT 扫描线扭曲现实，每次交互都感觉如指挥 2088 年的复古终端。

### 核心美学 DNA

**视觉语言**：高对比极繁主义配毫不妥协的霓虹饱和。无物含蓄。每个元素发光、变换或以数字能量脉动。设计拒绝极简主义，转而追求密集、分层的视觉效果，通过重叠渐变、辉光、扫描线与透视扭曲创造深度。

**情感基调**：怀旧而未来。同时复古（80 年代街机柜、VHS 磁带、早期 Windows UI）与前瞻（赛博朋克城市景观、全息界面、数字乌托邦）。气质梦幻、合成、略超现实——如导航一台来自过去未来愿景的计算机。

**设计支柱**：
1. **无限网格**：透视变换的线框网格向地平线退去，创造空间深度与标志性 outrun 公路感
2. **霓虹辉光至上**：热品红（#FF00FF）、电气青（#00FFFF）与日落橙（#FF9900）配激进投影与 box shadow 使元素显得发光
3. **CRT 扫描线与扭曲**：全局水平扫描线叠加与含蓄 RGB 色差模仿旧 CRT 显示器
4. **终端/命令行界面**：">" 符号前缀文字、等宽字体、配彩色圆点的窗口装饰、状态栏——一切引用 DOS 提示符与早期 GUI
5. **几何变换**：倾斜容器、旋转图标、透视网格——元素极少完美对齐；它们感觉动感且维度化
6. **渐变狂热**：处处多停靠点渐变——文字填充、背景、边框、辉光。尤其标志性日落渐变（黄 → 橙 → 粉 → 紫）

### 交互哲学

**悬停状态戏剧化**：按钮不只变色——它们去倾斜、爆发辉光、放大并反相色彩。图标旋转。卡片抬离页面。每次交互都是微事件。

**声音设计（视觉）**：若此设计有声，会是霓虹管嗡鸣、CRT 静电嗡嗡、复古合成器与 lo-fi 节拍。视觉设计通过脉动动画、故障效果与节奏重复（扫描线、网格图案）呼应此。

### "反模式"（此设计不是什么）
- **不扁平**：激进使用阴影、辉光、渐变与深度
- **不极简**：效果、边框、图案与叠加密集
- **不企业**：俏皮、艺术、实验——这是作品集，非银行
- **不柔和**：色彩 100% 饱和；对比极端

## 2. 设计 Token 系统（DNA）

### 色彩（仅深色模式）

**哲学**：最大饱和、高对比、纯数字原色。这些非含蓄品牌色——它们是暗虚空中发光的霓虹管。

*   **Background（虚空）**：`#090014` — 近黑配含蓄紫调。这是一切漂浮的无限数字空间。
*   **Foreground（镀铬文字）**：`#E0E0E0` — 浅银灰作正文。可读而复古。
*   **Card Background（玻璃面板）**：`rgba(26, 16, 60, 0.8)` 或 `#1a103c` — 半透明深紫。启用配 backdrop blur 的玻璃拟态。
*   **Primary Accent（热品红）**：`#FF00FF` — 纯品红。用于主 CTA、高亮、头像、特性图标、点缀边框。这是 THE hero 色。
*   **Secondary Accent（电气青）**：`#00FFFF` — 纯青。用于链接、聚焦环、次级边框、悬停状态、卡片标题辉光。完美补充品红。
*   **Tertiary Accent（日落橙）**：`#FF9900` — 鲜艳橙。克制用于特殊高亮、"太阳"渐变与吸睛元素。
*   **Border（默认）**：`#2D1B4E` — 柔暗紫。非交互边框与分隔符。
*   **Border（激活）**：`#00FFFF` 或 `#FF00FF` — 交互/悬停元素霓虹边框。

**渐变组合**：
- **日落渐变**：`linear-gradient(to right, #FF9900, #FF00FF, #00FFFF)` — 用于文字填充的签名 vaporwave 渐变
- **辉光渐变**：`linear-gradient(to bottom, #FF9900, #FF00FF)` — 用于浮动"太阳"背景元素
- **点缀条**：`linear-gradient(to right, #FF00FF, #00FFFF)` — 顶边框与点缀线的锐利渐变

### 字体

**字体哲学**：字体必须同时唤起复古计算终端与未来科幻界面。几何无衬线以增冲击，等宽以增真实。

*   **标题**：`"Orbitron", sans-serif`（字重：400, 500, 700, 900）
    - 几何、宽、未来字形
    - 用于：页面标题、区块标题、卡片标题、定价
    - 特征：偏好全大写、极端字重（black/900）、大尺寸紧凑字距
*   **正文/UI/代码**：`"Share Tech Mono", monospace`（字重：400）
    - 技术、终端般、固定宽度
    - 用于：正文、按钮、标签、输入框、状态文字
    - 特征：UI 元素大写、正文正常大小写、宽字距（tracking）

**字号阶梯与层级**：
- **Hero 标题**：`text-5xl` 至 `text-9xl`（80px-128px）配响应式缩放。跨多行以增戏剧。
- **区块标题**：`text-3xl` 至 `text-6xl`（30px-60px）。始终 bold/black 字重。
- **卡片/组件标题**：`text-2xl`（24px）。青色配文字辉光。
- **正文**：`text-lg` 至 `text-xl`（18px-20px）。慷慨行高以保可读。
- **UI 标签/按钮**：`text-sm` 至 `text-lg`、全大写、宽字距（`tracking-wider`、`tracking-widest`）。

**文字效果**：
- **标题辉光**：白文字 `drop-shadow-[0_0_10px_rgba(255,255,255,0.5)]`，渐变文字 `drop-shadow-[0_0_30px_rgba(255,0,255,0.6)]`
- **卡片标题辉光**：青标题 `drop-shadow-[0_0_5px_rgba(0,255,255,0.8)]`
- **渐变文字填充**：Hero 陈述用 `bg-gradient-to-r from-[#FF9900] via-[#FF00FF] to-[#00FFFF] bg-clip-text text-transparent`

### 圆角与边框

**边框哲学**：锐利、几何、高对比。边框是霓虹光管，非含蓄分隔符。

*   **圆角**：`rounded-none`（0px）为主。Vaporwave 激进几何棱角。偶尔 `rounded-full` 仅用于点/圆。
*   **边框宽度**：`border-2`（2px）为标准。强调或外容器用更重边框（`border-4`）。
*   **边框色**：
    - 默认/非激活：`#2D1B4E`（暗紫，含蓄）
    - 交互/悬停：`#00FFFF`（青）或 `#FF00FF`（品红）
    - 顶部点缀条：渐变或纯青（`border-t-2 border-t-[#00FFFF]`）
*   **多边框图案**：卡片常有彩色顶边框（`border-t-2`）加不同色含蓄侧边框以增分层效果

### 阴影与效果（辉光）

**效果哲学**：一切发光。阴影是彩色辉光，非暗投影。

*   **Box Shadows（霓虹辉光）**：
    - **品红辉光**：`shadow-[0_0_10px_#FF00FF]` 或 `shadow-[0_0_20px_#FF00FF]` 作强烈辉光
    - **青色辉光**：容器 `shadow-[0_0_20px_rgba(0,255,255,0.2)]`，输入框 `shadow-[0_0_15px_#00FFFF]`
    - **大面积辉光**：最终 CTA 等主容器 `shadow-[0_0_50px_rgba(0,255,255,0.2)]`
*   **文字阴影**（见字体章节）
*   **悬停状态辉光**：按钮与交互元素悬停时辉光强度戏剧性增加（基础辉光 2-3 倍）

### 纹理与背景图案

**图案哲学**：虚空绝不空。网格、扫描线、点与渐变层创造维度深度。

*   **透视网格地面**：
    ```css
    background-image: linear-gradient(transparent 95%, #FF00FF 95%), linear-gradient(90deg, transparent 95%, #FF00FF 95%);
    background-size: 40px 40px;
    transform: perspective(500px) rotateX(60deg) translateY(-100px) scale(2);
    mask-image: linear-gradient(to bottom, transparent, black);
    ```
    创造标志性退去网格效果
*   **浮动太阳**：巨型模糊渐变球（`h-[600px] w-[600px] blur-[100px] bg-gradient-to-b from-[#FF9900] to-[#FF00FF] opacity-20`）
*   **全局扫描线叠加**：
    ```css
    background: linear-gradient(rgba(18,16,20,0) 50%, rgba(0,0,0,0.25) 50%);
    background-size: 100% 4px;
    ```
    作为固定叠加应用于整页以增 CRT 效果
*   **RGB 色差**（含蓄）：`linear-gradient(90deg, rgba(255,0,0,0.06), rgba(0,255,0,0.02), rgba(0,0,255,0.06))`
*   **点阵图案**：区块背景 `radial-gradient(#FF00FF 1px, transparent 1px)` 配 `background-size: 20px 20px`
*   **图片渐变叠加**：通过 `bg-gradient-to-br from-[#FF00FF] to-[#00FFFF] opacity-20 mix-blend-overlay` 作双色调效果

## 4. 非通用"大胆"选择（"Wow"要素）

这些是防止 Vaporwave 风格显得通用的强制独特设计签名：

1.  **激进倾斜**：按钮与徽章用 `-skew-x-12` 变换，创造动态对角形状，悬停时去倾斜以增动感变形效果
2.  **全局 CRT 扫描线**：跨整个视口的固定叠加配水平线图案与 RGB 色差
3.  **透视网格背景**：多个区块用 CSS 透视变换创造标志性退去网格地面效果
4.  **渐变文字填充**：Hero 标题用多停靠点渐变背景裁切至文字（`bg-clip-text text-transparent`）
5.  **旋转图标容器**：特性图标置于 `rotate-45` 菱形容器中，悬停时旋转至 `rotate-90`
6.  **双边框图案**：卡片结合亮青顶边框与含蓄粉侧边框以增分层霓虹光管美学
7.  **终端/窗口装饰**：多种 UI 模式模仿复古 OS 界面（配彩色圆点的窗口标题栏、文件资源管理器布局、命令提示符）
8.  **巨型模糊太阳**：背景中巨型渐变球（`600px` 直径配 `blur-[100px]`）创造氛围深度
9.  **IRC 风格元素**：证言用聊天消息格式配 `<username>` 尖括号语法
10. **交替时间线布局**：How It Works 区块用交替左右布局配中央检查点线
11. **辉光悬停放大**：交互元素不只高亮——它们以 2-3 倍辉光强度爆发并触发色彩反相

## 5. 动画与运动

**哲学**：利落、机械、复古数字。如 CRT 显示器预热或旧计算机软件响应输入。

*   **过渡速度**：`duration-200 ease-linear` — 快、不自然、数字。无有机缓动曲线。
*   **悬停变换**：
    - 按钮：去倾斜、填充色、反相文字、爆发辉光
    - 卡片：上移（`-translate-y-2`）、增加阴影
    - 图标：旋转 45° 或缩放
    - 链接：加下划线、变色、加辉光
*   **持续动画**：
    - 信任指标：`animate-pulse` 以增注意
    - 终端光标：可加闪烁效果
    - 图标：占位符 `animate-pulse`
*   **变换原点**：透视网格上小心用 `transform-origin`（`top center`、`bottom center`）
*   **过渡类**：`transition-all`、`transition-colors`、`transition-transform` 视变化内容

## 6. 布局策略与间距

**容器宽度**：主要内容 `max-w-7xl`，定价 `max-w-6xl`，FAQ/Final CTA `max-w-4xl`，Hero `max-w-5xl`

**间距系统**：
*   **区块内边距**：`py-20 sm:py-32`（80px-128px 垂直节奏）
*   **组件间隙**：网格 `gap-8`（32px），更大间距 `gap-12`（48px）
*   **内部内边距**：卡片 `p-6` 或 `p-8`，容器移动 `px-4`
*   **外边距**：慷慨——标题有 `mb-8` 至 `mb-20` 视尺寸

**网格使用**：
*   特性：`grid-cols-1 md:grid-cols-3`
*   数据：`grid-cols-1 md:grid-cols-2 lg:grid-cols-4`
*   博客：`grid-cols-1 md:grid-cols-3`
*   收益：`grid-cols-1 md:grid-cols-2 lg:grid-cols-3`
*   定价：`grid-cols-1 md:grid-cols-3`

**Z-Index 分层**（后至前）：
1. 背景网格（固定，`z-0`）
2. 浮动太阳渐变（固定）
3. 区块背景
4. 内容（导航/区块 `z-10`）
5. 扫描线叠加（固定，`z-50`）

## 7. 响应式策略

**断点**：移动优先方法用 `sm:`、`md:`、`lg:` 前缀

**移动适配**（< 640px）：
*   **字体**：标题缩小 1-2 级（如 `text-5xl` 替代 `text-8xl`）
*   **间距**：区块内边距从 `py-32` 减至 `py-20`，外边距从 `mb-20` 减至 `mb-12`
*   **网格**：堆叠为单列（`grid-cols-1`）
*   **按钮**：Hero 全宽 CTA 按钮垂直堆叠
*   **时间线**：左对齐配偏移而非交替布局
*   **边框**：保持霓虹边框（氛围必需）
*   **辉光效果**：略减强度以防小屏过载
*   **网格背景**：保持透视网格但简化（它们增添必需氛围）
*   **触控目标**：按钮通过 `h-12` 与 `h-14` 尺寸保持最小 44px 高度

**平板**（640px - 1024px）：
*   **网格**：常 2 列后跳至 3/4
*   **字体**：中档尺寸
*   **导航**：平板显示完整菜单

**关键**：Vaporwave 美学必须在移动端存活。霓虹辉光、边框与网格背景即使小屏也不可妥协。
