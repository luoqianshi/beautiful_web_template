01

设计哲学 · 深夜实验室

这是一种深色模式下的现代科技风格。它像一间深夜的实验室：黑色背景是安静的实验台，荧光绿是示波器上跳动的信号，白色文字是清晰的读数，灰色文字是辅助的标注。

不是消费级 UI

拒绝温暖柔和、色彩斑斓或厚重拟物。强调功能优先、结构清晰、交互反馈即时，呈现冷静、工程师、未来感的气质。

结构即美学

通过极高对比度让信息一目了然。胶囊形按钮与圆角卡片带来恰到好处的亲和力，在冷峻中保留人性的温度。

即时反馈

几乎所有可交互元素在 Hover 或 Focus 时都会产生绿色边框或绿色阴影，让用户始终感受到系统的响应与活力。

04

效果与动画 · 流动与呼吸

动画是 TRAE Dark Tech 的签名特征。流动的粒子网络背景、滚动触发的入场效果、即时的 Hover 反馈，共同营造出沉浸式的科技体验。

粒子网络背景

Canvas 2D 全屏绘制 80-120 个绿色粒子，距离小于阈值时自动连线，鼠标靠近产生交互吸引，是整套风格最具辨识度的视觉元素。

滚动入场动画

元素进入视口时触发 opacity 0→1 与 translateY(30px)→0 的过渡，持续 0.6s。进度条在可见时从 0 宽度扩展到目标值，持续 1.5s。

Hover 光晕反馈

按钮悬停时上移并扩散绿色阴影，卡片悬停时边框变绿并产生柔和辉光，输入框聚焦时获得绿色边框与外发光，反馈即时且一致。
