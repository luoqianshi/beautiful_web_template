## 布局创意 (Layout Ideas)

### Hero

居中布局配巨型标题。文字后方环境辉光创造深度。信任徽章浮于上方配含蓄辉光与脉动圆点。CTA 并排，主按钮悬停时琥珀辉光效果。背景有极含蓄径向渐变暖意。

### Stats

横向条带配玻璃效果背景。数据间含蓄垂直分隔符（1px，低不透明度）。数字用展示字体配琥珀点缀。含蓄上下边框。

### Product Detail

双栏布局。左侧环境辉光球装饰。文字内容左栏右对齐。右栏浮动玻璃卡片含抽象 UI 模型。

### Features

干净 3 列玻璃卡片网格。图标置于琥珀色调圆圈中。悬停显现含蓄辉光。首个特性可跨 2 列以强调。一致卡片高度。

### How It Works

横向编号步骤。大号圆圈数字配琥珀填充。步骤间连接线（含蓄、虚线）。下方各配玻璃效果卡片。

### Benefits

分割布局，左侧大型环境球。右侧收益为干净列表配琥珀勾选标记。项目间慷慨间距。

### Pricing

3 列玻璃卡片。高亮档位有琥珀边框辉光、'Popular' 徽章，并略大（桌面 scale-105 + translate-y-4 以突出）。价格大号展示字体。特性列表配含蓄勾选标记。

### Testimonials

错落 3 列布局。玻璃卡片左边缘含蓄琥珀点缀线。头像图片配 ring 边框。引文斜体。

### Faq

干净手风琴配 plus/minus 图标。问题 medium 字重，回答 regular。项目间含蓄分隔符。无背景。

### Blog

3 列网格，图片玻璃卡片效果。悬停卡片轻微上浮。日期/作者柔和文字。干净字体层级。

### Footer

多列配含蓄顶边框。左侧 logo 与描述。右侧导航组。社交图标为含蓄幽灵按钮。非常干净、极简。

## 完整提示词 (Full Prompt)

# 设计风格：Minimalist Dark

## 设计哲学

### 核心原则

**氛围深度。** Minimalist Dark 不通过色彩饱和或复杂图案创造视觉趣味，而通过精心编排的黑暗层次。多层石板与炭灰相互堆叠，配温暖琥珀点缀如夜中余烬般发光。设计呼吸——慷慨的留白（或更确切说"暗空间"）给每个元素存在空间。

### 视觉气质

**情感关键词**：氛围感、精致、宁静、高端、夜行、考究、宽敞、暖冷对比、空灵、沉稳

这是以下事物的视觉语言：
- 高端深色模式应用（Linear、Raycast、Arc）
- 高端开发者工具（Vercel、Railway）
- 夜间的奢华科技产品
- 你会在凌晨 2 点使用的精美设计应用
- 精工细作软件的宁静自信

设计感觉如在夜晚完美照明的房间中工作——一切可见，无物伤眼，有平静专注之感。

### 此设计不是什么

- 纯黑（改用丰富石板色调）
- 刺眼或高对比
- 多彩或鲜艳
- 冷或无菌
- 扁平或无阴影
- 类似 Minimalist Modern（无蓝渐变、非处处 rounded-lg）
- 类似 Minimalist Monochrome（有色彩点缀、更柔边缘、非编辑风）

### Minimalist Dark 的 DNA

#### 1. 分层石板调色板
非纯黑——丰富石板色调（#0A0A0F 最深、#12121A 卡片背景、#1A1A24 抬升表面）。每层含蓄不同，通过黑暗本身创造深度。

#### 2. 温暖琥珀点缀
单一暖点缀色（#F59E0B / amber-500）对冷暗色调创造美丽对比。克制用于交互元素、高亮与焦点。温暖防止设计感觉冰冷。

#### 3. 环境辉光效果
关键元素后方的柔和模糊辉光创造氛围深度。非刺眼投影——想象环境光透过黑暗渗出。应用于按钮悬停（0_0_20px 配 0.4 不透明度）、Hero 徽章、证言点缀线与装饰球。辉光含蓄但对氛围质量至关重要——它们创造"黑暗中的光"之感。

#### 4. 玻璃效果卡片
卡片用半透明背景配含蓄 backdrop blur。边框不透明度低（10-15%）。这创造分层、浮动效果而无刺眼边缘。

#### 5. 几何无衬线字体
Space Grotesk 用于展示，Inter 用于正文。干净、几何字形，感觉现代而技术。通过尺寸与字重建立强层级，非色彩变化。

#### 6. 慷慨呼吸空间
极宽敞布局。大区块内边距。内容不拥挤——漂浮于空间中。此呼吸空间对高端感至关重要。

#### 7. 含蓄边框
边框存在但极含蓄——通常 1px 配 10-20% 不透明度。它们定义边缘而不吸引注意。无厚重边框。

### 与其他极简风格的区别

| 方面 | Minimalist Modern | Minimalist Monochrome | Minimalist Dark |
|------|-------------------|----------------------|-----------------|
| 模式 | 浅 | 浅 | **深** |
| 背景 | 近白 | 纯白 | 深石板（#0A0A0F） |
| 点缀 | 蓝渐变 | 无（仅黑） | 暖琥珀（#F59E0B） |
| 字体 | 无衬线 + 展示衬线 | 全衬线 | 几何无衬线 |
| 角 | 圆（lg, xl） | 锐（0px） | 柔圆（md, lg） |
| 深度 | 阴影 + 辉光 | 扁平、无阴影 | 环境辉光 + 玻璃 |
| 感觉 | 活力、当代 | 编辑、冷峻 | 氛围、宁静 |
| 边框 | 含蓄 | 重黑线 | 极含蓄、低不透明度 |

## 设计 Token 系统

### 色彩（深石板 + 琥珀）

```
background:       #0A0A0F (深石板 - 近黑但更暖)
backgroundAlt:    #12121A (略抬升表面)
foreground:       #FAFAFA (近白文字)
muted:            #1A1A24 (卡片背景、抬升表面)
mutedForeground:  #71717A (次要文字 - zinc-500)
accent:           #F59E0B (Amber-500 - 温暖、发光)
accentForeground: #0A0A0F (琥珀上深色文字)
accentMuted:      rgba(245, 158, 11, 0.15) (琥珀辉光背景)
border:           rgba(255, 255, 255, 0.08) (极含蓄边框)
borderHover:      rgba(255, 255, 255, 0.15) (悬停边框)
card:             rgba(26, 26, 36, 0.6) (半透明卡片)
cardSolid:        #1A1A24 (实心卡片背景)
ring:             #F59E0B (聚焦环)
```

### 字体

**字体栈**：
- **展示/标题**：`"Space Grotesk", system-ui, sans-serif` — 几何、技术、独特
- **正文**：`"Inter", system-ui, sans-serif` — 干净、高可读
- **等宽**：`"JetBrains Mono", monospace` — 用于代码、标签、元信息

**字号阶梯**：
```
xs:   0.75rem   (12px)
sm:   0.875rem  (14px)
base: 1rem     (16px)
lg:   1.125rem (18px)
xl:   1.25rem  (20px)
2xl:  1.5rem   (24px)
3xl:  2rem     (32px)
4xl:  2.5rem   (40px)
5xl:  3.5rem   (56px)
6xl:  4.5rem   (72px)
7xl:  6rem     (96px)
```

**字距**：
- 标题：`tracking-tight`（-0.025em）
- 正文：`tracking-normal`（0）
- 标签/等宽：`tracking-wide`（0.025em）

### 圆角

```
sm:   0.375rem (6px)
md:   0.5rem   (8px) - 大多数元素默认
lg:   0.75rem  (12px) - 卡片、较大容器
xl:   1rem     (16px) - Hero 元素、大卡片
2xl:  1.5rem   (24px) - 特殊装饰元素
full: 9999px   - 胶囊、头像
```

比锐角更柔，但不如 Modern 戏剧化圆。

### 阴影与辉光

```
// 含蓄抬升阴影
sm:   0 1px 2px rgba(0, 0, 0, 0.3)
md:   0 4px 6px rgba(0, 0, 0, 0.3)
lg:   0 10px 15px rgba(0, 0, 0, 0.3)
xl:   0 20px 25px rgba(0, 0, 0, 0.4)

// 环境辉光（签名效果）
glowSm:   0 0 20px rgba(245, 158, 11, 0.15)
glowMd:   0 0 40px rgba(245, 158, 11, 0.2)
glowLg:   0 0 60px rgba(245, 158, 11, 0.25)

// 高亮元素边框辉光
borderGlow: 0 0 0 1px rgba(245, 158, 11, 0.3), 0 0 20px rgba(245, 158, 11, 0.15)
```

### 纹理与图案

**含蓄噪点叠加**（极低不透明度）：
```css
background-image: url("data:image/svg+xml,...noise...");
opacity: 0.02;
```

**径向渐变氛围**（用于区块背景）：
```css
background: radial-gradient(ellipse at top, rgba(245, 158, 11, 0.03) 0%, transparent 50%);
```

**含蓄网格**（可选，用于特定区块）：
```css
background-image: linear-gradient(rgba(255,255,255,0.02) 1px, transparent 1px),
                  linear-gradient(90deg, rgba(255,255,255,0.02) 1px, transparent 1px);
background-size: 40px 40px;
```

### 按钮

**主按钮**：
```
- 背景：#F59E0B (琥珀)
- 文字：#0A0A0F (深)

- Focus-visible：ring-2 ring-[var(--accent)] ring-offset-2
- 过渡：all 200ms ease-out
```

**次/描边按钮**：
```
- 背景：透明
- 文字：#FAFAFA
- 边框：1px solid rgba(255,255,255,0.15)
- 悬停：bg-white/5，border-white/25
- 激活：scale-[0.98]
- Focus-visible：同主
```

**幽灵按钮**：
```
- 背景：透明
- 文字：#FAFAFA
- 边框：无
- 悬停：bg-white/5
- 激活：scale-[0.98]
- Focus-visible：同主
```

### 卡片（玻璃效果）

**标准卡片**：
```css
background: rgba(26, 26, 36, 0.6);
backdrop-filter: blur(8px);
border: 1px solid rgba(255, 255, 255, 0.08);
border-radius: 12px;
transition: all 300ms ease-out;
```

**悬停状态**（交互时）：
```css
border-color: rgba(255, 255, 255, 0.15);
background: rgba(26, 26, 36, 0.8);
transform: scale(1.02);
box-shadow: 0 10px 15px rgba(0, 0, 0, 0.3);
```

**高亮卡片**（如精选定价档位）：
```css
/* 同标准加： */
border: 1px solid rgba(245, 158, 11, 0.2);
box-shadow: 0 0 0 1px rgba(245, 158, 11, 0.2), 0 0 30px rgba(245, 158, 11, 0.15);
/* 桌面端也可用 scale(1.05) 与 translate-y 以强调 */
```

### 输入框

```
- 背景：rgba(26, 26, 36, 0.6)
- Backdrop-filter：blur(8px)
- 边框：1px solid rgba(255,255,255,0.08)
- 圆角：rounded-lg
- 高度：h-11 (44px 以保正确触控目标)
- 文字：#FAFAFA
- 占位符：#71717A
- 聚焦：border-amber-500/50，ring-2 ring-amber-500/20，shadow-[0_0_20px_rgba(245,158,11,0.1)]
- 过渡：all 200ms
```

## 效果与动画

**运动哲学**：平滑含蓄配愉悦微交互

- **过渡**：200-300ms，ease-out（卡片用 300ms 以更流畅）
- **悬停效果**：
  - 卡片：含蓄缩放（scale-[1.02]）、边框提亮、辉光增强
  - 按钮：辉光增强（阴影强度至 0.4）、亮度提升
  - 链接：focus-visible 时色彩转点缀
- **激活状态**：按钮含蓄按压效果（scale-[0.98]）
- **动画**：
  - Hero 徽章脉动圆点（animate-pulse 配辉光）
  - FAQ 手风琴平滑高度过渡（max-h 配透明度淡入）
- **不要**：弹跳动画、戏剧变换
- **要**：轻柔淡入、柔和辉光、平滑状态变化、含蓄缩放

```css
/* 卡片 */
transition: all 300ms ease-out;

/* 按钮与快速交互 */
transition: all 200ms ease-out;
```

**环境球**（装饰背景元素）：
- 大型模糊圆配琥珀辉光
- 极低不透明度（0.02-0.04）
- 战略性定位（顶部居中、右下作固定背景）
- 模糊值：100px-150px 以柔和弥散光
- 响应式：移动端更小尺寸以保性能（移动 h-[400px] vs 桌面 h-[600px]）

## 响应式策略

**移动端适配**：
- 保持深色调色板与暖点缀——美学不妥协
- 平滑缩放字体：`text-4xl sm:text-5xl md:text-6xl lg:text-7xl`
- 列垂直堆叠（双栏布局 `lg:grid-cols-2`）
- 减小环境辉光球尺寸以保性能（但保留！）
- 保持慷慨垂直间距（`py-24 md:py-32 lg:py-40`）
- 触控目标：最小 44px 高（按钮用 h-11 或 h-12）
- 移动端隐藏导航（`hidden md:flex`），暗示汉堡菜单
- 所有悬停状态在触控设备上也作激活状态
- 保持玻璃效果（backdrop-blur 在现代移动端高性能）

**关键原则**：氛围质量必须在移动端存活。这不是"移动简化"版——是相同高端体验，仅适配屏幕尺寸。

## 可访问性

**对比**：
- 主要文字（#FAFAFA）于背景（#0A0A0F）：18.4:1 比例（超过 AAA）
- 柔和文字（#71717A）于背景：4.9:1 比例（达 AA）
- 琥珀点缀在深浅上下文均可读

**聚焦状态**：
所有交互元素用 `focus-visible` 有清晰可访问聚焦状态：

**按钮**：
```css
focus-visible:outline-none
focus-visible:ring-2
focus-visible:ring-[var(--accent)]
focus-visible:ring-offset-2
focus-visible:ring-offset-[var(--background)]
```

**链接**（导航、页脚等）：
```css
focus-visible:text-[var(--accent)]
focus-visible:outline-none
```

**输入框**：
```css
focus:border-[var(--accent)]/50
focus:outline-none
focus:ring-2
focus:ring-[var(--accent)]/20
```

琥珀点缀色一致用于所有聚焦指示器，保持品牌连贯同时确保可见。

## 标志性选择（不可妥协）

1. **分层黑暗**：至少 3 种不同暗调可见（#0A0A0F → #12121A → #1A1A24）
2. **温暖琥珀点缀**：无冷蓝——#F59E0B 琥珀创造签名温暖
3. **环境辉光效果**：
   - Hero 徽章：含蓄辉光 + 脉动圆点
   - 按钮悬停：0_0_20px 辉光配 0.4 不透明度
   - 证言点缀线：柔和辉光
   - 背景环境球：超大模糊（100-150px）
4. **玻璃效果卡片**：半透明（0.6 不透明度）配 backdrop blur（8px）
5. **慷慨间距**：py-24 md:py-32 lg:py-40 区块感觉宽敞，非局促
6. **含蓄边框**：rgba(255,255,255,0.08) — 仅 8% 不透明度，绝不刺眼
7. **几何字体**：Space Grotesk 标题、Inter 正文、JetBrains Mono 标签
8. **氛围背景**：固定环境球 + 含蓄噪点纹理（0.015 不透明度）
9. **微交互**：
   - 卡片悬停放大（1.02）
   - 按钮激活缩小（0.98）
   - 平滑 FAQ 手风琴配高度 + 透明度过渡
   - 所有聚焦状态用琥珀点缀

## 成功是什么样

成功实现的 Minimalist Dark 设计应感觉如：
- 夜间使用 Linear 或 Raycast
- 高端开发者工具的营销网站
- 为专注与宁静设计的软件
- 暗室中发光的温暖光线

不应感觉如：
- 色彩反转的通用深色主题
- 刺眼或高对比
- 冷或不受欢迎
- Minimalist Modern 的深色版复制
- 仅"深色模式"——它应有自己的个性
