How it works

Three steps to ethereal

From blank canvas to a weightless interface.

1

Layer the mesh

Drop in animated gradient blobs behind a black canvas. Color comes from the background.

2

Add the glass

Apply 50px blur and 180% saturation to translucent cards. Let the mesh bleed through.

3

Constrain and ship

Use tight max-widths to make content feel precious. Add slow, smooth transitions.

FAQ

Frequently asked

Everything you need to know about going ethereal.

Does glassmorphism hurt performance?
Modern browsers handle backdrop-filter efficiently with GPU acceleration. We recommend limiting blur to above-the-fold glass elements and using simpler backgrounds deeper in the page. Aura is optimized for 60fps on mid-range hardware.
Is it accessible?
Yes, when done carefully. We ensure sufficient contrast for text on glass by using off-white (#F5F5F7) on dark mesh backgrounds. All interactive elements have visible focus rings. The mesh is decorative and hidden from screen readers.
Can I customize the mesh colors?
Absolutely. The mesh is built from five independently animated blobs. You can change their colors, sizes, positions, and animation delays. Studio plan includes a visual mesh theme builder.
Does it work in light mode?
Aura is dark-mode-first by design — the mesh glows most beautifully against black. A light variant is possible by using darker mesh blobs and lighter glass tints, but we recommend embracing the dark canvas.
