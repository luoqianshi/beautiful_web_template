man term.sys

// --help for available flags
--speed SECTION 1

Instant Execution

Compiled to native bytecode with zero virtual machine overhead. Commands return before you finish typing the next one.

$ term.sys run --fast
--security SECTION 2

Encrypted By Default

End-to-end AES-256 with rotating keys. Every packet signed. Every session ephemeral. No plaintext crosses the wire.

$ term.sys encrypt --on
--scriptable SECTION 3

Pipe Everything

STDOUT composable with grep, awk, sed, jq. Build pipelines that span services. The shell is the API.

$ term.sys get | jq '.[]'
--offline SECTION 4

Local-First

Full functionality without a network connection. Sync when reconnected. Your data never leaves the machine unless you say so.

$ term.sys sync --later
--audit SECTION 5

Immutable Logs

Append-only audit trail with cryptographic chaining. Every action traceable. Nothing retractable. Total accountability.

$ term.sys log --tail
--extend SECTION 6

Plugin Architecture

Write extensions in any language that speaks STDIN/STDOUT. Distribute via git. No package manager lock-in.

$ term.sys plug add ./ext

man term.sys(7)

// help --topic faq
TERM.SYS(7) Miscellaneous Information Manual TERM.SYS(7)
Is term.sys open source?▶
Yes. The core runtime is MIT-licensed and available on our git server. Enterprise plugins are source-available under a commercial license. Fork it, audit it, contribute back.
Can I run it without a GUI?▶
Absolutely. term.sys is designed for headless operation. SSH into a server, pipe commands, read STDOUT. A GUI is a convenience, never a requirement.
Does it work on Windows?▶
Yes — via WSL2, Cygwin, or the native binary we ship for x64 and ARM64. PowerShell integration is supported through our PSReadLine-compatible module.
How is data encrypted at rest?▶
AES-256-GCM with per-record keys derived from a master key sealed by your OS keychain (Keychain, Secret Service, or DPAPI). Keys never touch disk in plaintext.
What is the blinking cursor for?▶
It is the heartbeat of the interface. When the cursor blinks, the system is alive and listening. It is the only animation that matters.
